# Project 4: Multi-Source RAG Chatbot

    Build a web app where users upload PDFs + paste URLs, and a RAG pipeline answers questions over that combined knowledge base using embeddings + vector search. Format: WebApp (Streamlit or FastAPI + simple frontend).

    Covers: Async execution concepts (async API calls while UI stays live), pydantic schemas for validating query inputs, API interactions at depth (chunking, embedding endpoints), environment management.

    Swiss value: RAG is the dominant enterprise AI pattern in Swiss consulting (Accenture Zurich, Deloitte Geneva). This is a direct portfolio signal for AI Engineer roles.
