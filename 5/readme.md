# Project 5: Automated ML Pipeline with Logging + Alerting

    Build a pipeline that pulls data on a schedule, retrains a sklearn model, evaluates metrics, logs everything with the logging module, and sends an email alert if accuracy drops below a threshold. Format: CLI + scheduled script.

    Covers: Structured logging (handlers, formatters, logger.error vs logger.info), try/except/finally defensive blocks, file system automation (pathlib, shutil for archiving old model versions), virtual environment and dependency concepts (poetry lockfile understanding).

    Swiss value: This is a junior MLOps task verbatim. Swiss precision culture values observability — an unmonitored pipeline is an untrustworthy pipeline.
