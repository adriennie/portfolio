# Project 6: Pydantic-Validated AI Form Processor

    Build a FastAPI backend where users submit structured forms (job applications, patient intake, contract data), a pydantic schema validates and sanitizes inputs, and Claude API generates a structured analysis/summary of each submission. Format: WebApp (FastAPI + minimal HTML frontend).

    Covers: Pydantic schemas and type validation deeply (field validators, model nesting), OOP paradigms (service class architecture), async/await concepts (FastAPI is async-native), JSON response parsing.

    Swiss value: Swiss enterprise software requires airtight data contracts. This directly maps to what SAP Switzerland, ServiceNow, and Swiss Re AI teams build.
