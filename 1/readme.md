# AI News Summarizer CLI

Build a CLI tool that fetches RSS feeds, sends article text to Claude/OpenAI API, and prints clean summaries. Format: CLI.

Covers: API interactions (requests, bearer tokens, JSON parsing), environment management (dotenv, os), basic error catching (try/except).

Swiss value: Shows you can integrate external AI APIs securely without leaking credentials — day-one junior skill at Zurich fintech/AI firms.
